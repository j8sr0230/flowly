flowly.core.hashable module¶

class flowly.core.hashable.Hashable(uuid=None)¶

Bases: object

A base class for objects that can be uniquely identified and compared using a UUID.

The Hashable class generates a UUID for each instance, which is used to make instances hashable and comparable. If a UUID is not provided during initialization, a new UUID is generated automatically.

Variables:

_uuid (UUID) – The unique identifier for the instance. This is either provided during initialization or generated automatically using uuid1().
